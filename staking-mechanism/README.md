# Staking Mechanism

The staking system is presented as a core part of the Tetreum economic model.

Users can lock TET for predefined periods and receive rewards in return. The system is described as non-custodial and smart-contract based, allowing users to verify the logic and outcomes on-chain.

## Design goals

- Encourage long-term participation
- Provide predictable reward mechanics
- Support transparent and verifiable on-chain behavior
- Reduce reliance on intermediaries

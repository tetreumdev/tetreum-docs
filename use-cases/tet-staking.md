# TET Staking

The whitepaper describes staking as a way for token holders to lock TET, participate in the ecosystem, and earn rewards.

## Staking themes in the source

- Reward structures based on duration and amount
- Lock periods with multiple options
- Accountability through penalties for inactivity or malicious behavior

The document positions staking as both an incentive layer and a way to align long-term participation with network growth.

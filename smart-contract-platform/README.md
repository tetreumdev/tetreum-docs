# Smart Contract Platform

Tetreum is described as fully compatible with the Ethereum Virtual Machine, allowing developers to deploy and interact with smart contracts using established Ethereum tools and workflows.

## Platform highlights

- Solidity-based development
- Support for common Ethereum tooling
- JSON-RPC connectivity
- REST API access for blockchain data
- Developer-friendly portability for existing Ethereum dApps

This section is the developer-facing foundation of the documentation.

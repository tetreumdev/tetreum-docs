# Import Instructions

## Best option: Git Sync

GitBook works best with Markdown repositories.

1. Upload this folder to a new GitHub or GitLab repository.
2. In GitBook, create a new space.
3. Choose **Git Sync** and connect the repository.
4. GitBook will use:
   - `README.md` as the homepage
   - `SUMMARY.md` as the sidebar
   - `.gitbook.yaml` as the content configuration

## Important note

GitBook's standard file import UI has a simultaneous upload limit for imported pages. This package is intended primarily for **Git Sync**, which is the most reliable way to import the full documentation tree.

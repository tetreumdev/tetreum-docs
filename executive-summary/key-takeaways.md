# Key Takeaways

- **Fixed supply:** TET has a maximum supply of 1,000,000,000 tokens.
- **Consensus:** Tetreum uses Proof of Authority for efficient, eco-friendly block validation.
- **Staking:** Users can lock tokens and earn rewards.
- **Governance:** Governance participation is designed to be part of the ecosystem from launch.
- **Developer support:** The network is designed for EVM compatibility and Ethereum-style tooling.
- **Fast transactions:** Finality is described as taking seconds rather than minutes.
- **Utility-first design:** The protocol is intended to support real-world applications and decentralized infrastructure.

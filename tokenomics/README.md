# TET Tokenomics

Tetreum's token model is presented as a fixed-supply system designed for long-term sustainability.

## Core token facts

- **Token name:** TET
- **Maximum supply:** 1,000,000,000 TET
- **Token contract address:** `0x8C8407a0Eca05B481bFb13C0E5061b3Be775Be55`

The token economy is intended to support network development, stakeholder incentives, liquidity, and community growth.
